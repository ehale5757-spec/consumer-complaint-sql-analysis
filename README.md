Consumer Complaint Data Analysis (SQL)
Objective

This project analyzes U.S. consumer complaint data to identify trends in financial issues, regional differences, and company-level complaint patterns.

Dataset

Data sourced from the Consumer Financial Protection Bureau (CFPB), including fields such as:
State
Product
Issue
Company
Date received

Tools Used 
SQL (SQLite)
DBeaver

Key Questions
Which states have the most complaints?
What issues are most common?
Which companies receive the most complaints?
How have complaints changed over time?

Key Insights
Complaint volume is concentrated in a small number of states, suggesting regional disparities in consumer issues.
A small number of companies account for a large share of complaints, indicating recurring service concerns.
Certain financial products generate significantly more complaints, highlighting areas of consumer risk.
Trends over time show changes in complaint volume, possibly reflecting shifts in consumer awareness or market conditions.
